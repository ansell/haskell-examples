haskell-examples
================

Examples of how to use/build haskell programs

https://lotz84.github.io/haskellbyexample/ex/values

Dependencies
============

* GHC : apt-get install ghc

Running
=======

As a script:

    runhaskell src/values.hs 

Compiling:

    ghc src/values.hs -o dist/build/values
    dist/build/values
